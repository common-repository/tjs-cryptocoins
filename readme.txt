Receive Bitcoin or Gulden donations with this QR Code Widget.

== Description ==

With this plugin you're able to add your Bitcoin or Gulden wallet address as QR-code to the sidebar of your website. 
The address you fill in will be converted automatically to a QR-code, which allows visitors to donate Bitcoins or Guldens to the desired address. 
Just fill-in a desired title, message and your wallet address and you're good to go!
The plugin displays a title, message, QR-code, an auto-select address input field and a button to allow users to buy Bitcoins or Guldens.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Go to Apperance->Widgets to drag and drop the widget in a desired sidebar
1. Fill in the desired data and don't forget your cryptocoin wallet address
1. Save your settings
